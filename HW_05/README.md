# 應用程式設計 - 課後實作 05

### 問題描述 - 1
寫出一程式，輸入正整數a、b (a <= b)，輸出 a、 b之間 4 或 9 之倍數(包含 a、b)，並算出「倍數數量」與「倍數總和」。

### 範例輸入
```
1
10
```
### 範例輸出
```
4, 8, 9，共有 3 個，總和為 21
```

### 問題描述 - 3
請統計附檔內 (snow-white.txt) 英文字母a到z出現的次數(不計大小寫)，並將統計結果列印文字輸出以及繪製為長條圖。 橫軸是英文字母 a - z，縱軸是出現的次數。
### 範例輸入
```
無
```
### 範例輸出
```
{'a': 100, 'b': 121, 'c': 12, ......, 'z': 2}
```
