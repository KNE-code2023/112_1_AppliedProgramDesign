# 應用程式設計 - 課後實作 10

## 問題描述
任選之前的作業程式五項，註記為 A, B, C, D, E。  
設計一組程式，可以顯示文字型的選單，選單上有 A-E 與 Z 六個選項，如輸入選項 A，程式會啟動 A 程式，執行完畢後會回到選單．如輸入選項 Z，則退出選單。
### 如何使用
```bash
python main.py
```
## 目錄結構
```bash
Assignment_10/
│
├─ code/
│  ├─ A.py
│  ├─ B.py
│  ├─ C.py
│  ├─ D.py
│  └─ E.py
│
├─ config/
│  └─ config.json
│
├─ main.py
└─ README.md
```
* `code` ： 存放 A-E.py
* `config` ： code 資料夾內的程式描述
* `main.py` ： 主程式
