# 應用程式設計 - 課後實作 10

## 問題描述
任選之前的作業程式五項，註記為A, B, C, D, E，設計一組程式，可以顯示文字型的選單，選單上有六個選項，分別是 A-E 與 Z，輸入選項 x，程式會啟動作業 x 的程式，執行完畢後會回到選單．如輸入選項 Z，則結束程式． 
### 如何使用
```bash
python main.py
```
